## Usage

1. Open `00_GLOBAL_DESIGN_LOCK.md` and copy the lock block.
2. Open one phase file, then paste the lock block first and the phase prompt next into Stitch.
3. Generate phases in order 1 to 6 for visual continuity.
4. If style drift appears, paste the global lock block again before generating the next phase.
5. Use `07_OPTIONAL_MASTER_BATCH_PROMPT.md` when you want all screens in one Stitch run.
