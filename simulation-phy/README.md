# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/810e3c5e-2e40-4da9-9011-d680ae7959e9

## Scope

This project is a learning simulation only. It must not include login, signup, profile, or account-menu features.

Educator mode is a simulation mode for teaching workflows and is not identity-gated.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
